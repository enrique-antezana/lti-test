üìö LTI 1.3 Launch Flow

  1. Platform Initiation: LTI platform sends login request to /login
  2. OIDC Authentication: Tool authenticates with platform using OpenID Connect
  3. Redirect Back: Tool redirects back to platform with authentication tokens
  4. Message Launch: Platform sends launch message to /launch
  5. Content Display: Tool validates launch and displays content

üîê Security Features

  • JWT Validation: All messages are cryptographically signed
  • Nonce Verification: Prevents replay attacks
  • Deployment ID Validation: Ensures tool is launched from authorized context
  • Scope Checking: Verifies requested permissions
