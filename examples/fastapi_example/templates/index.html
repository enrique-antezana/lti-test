üöÄ Getting Started

To run this example:

  1. Install dependencies: pip install -r requirements.txt
  2. Generate RSA keys: openssl genrsa -out private.key 2048
  3. Extract public key: openssl rsa -in private.key -pubout -out public.key
  4. Update configuration in main.py
  5. Run the server: python main.py
